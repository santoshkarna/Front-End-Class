Directives:

Directives are classes in Angular that allow us to manipulate the DOM or extend the behavior of elements and components. There are three types of directives: 1.Component(technically a directive with a template) 2.Structural directives: modify layout 3.Attribute Directives: modify appearance/behavior
Structural directives:

Common Structural Directives:

  • *ngIf => Adds/removes an element based on condition
  • *ngFor => Iterates over a collection
  • *ngSwitch => Switch-case conditional rendering
